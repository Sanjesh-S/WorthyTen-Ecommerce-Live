# Enable GitHub Pages - Quick Guide

## ✅ Your repository is ready! Code has been pushed successfully.

Now follow these 5 simple steps to make your site live:

### Step 1: Go to Repository Settings
1. Open your browser and go to: https://github.com/Sanjesh-S/WorthyTen-Ecommerce-Live
2. Click the **"Settings"** tab (near the top right)

### Step 2: Find Pages Settings
1. In the left sidebar, scroll down and click **"Pages"**
   (It's under the "Code and automation" section)

### Step 3: Configure GitHub Pages
1. Under **"Source"** (also called "Build and deployment"):
   - Click the dropdown that says "None"
   - Select **"Deploy from a branch"**

2. Under **"Branch"**:
   - First dropdown: Select **"main"**
   - Second dropdown: Select **"/ (root)"**
   - Click the **"Save"** button

### Step 4: Wait for Deployment
- GitHub will show a message: "Your site is being built"
- Wait 2-3 minutes for the deployment to complete
- Refresh the page to see the status

### Step 5: Access Your Live Site
Your website will be live at:
**https://sanjesh-s.github.io/WorthyTen-Ecommerce-Live/**

## ❓ Troubleshooting

**If the site doesn't load:**
- Wait 5 minutes (first deployment can take time)
- Hard refresh: Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac)
- Check GitHub Pages settings to ensure "main" branch is selected

**If you see 404 errors:**
- Make sure index.html is in the root directory (it is ✅)
- Verify the branch name is "main" not "master"
