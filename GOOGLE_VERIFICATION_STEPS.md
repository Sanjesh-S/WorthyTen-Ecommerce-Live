# Google Search Console Verification & Security Review - Step by Step

## 📋 **Step 1: Get HTML Tag from Google Search Console**

1. Go to: https://search.google.com/search-console
2. If you see the verification modal, click **"Use another method"**
3. Select **"HTML tag"** method
4. You'll see a meta tag like this:
   ```html
   <meta name="google-site-verification" content="YOUR_VERIFICATION_CODE_HERE" />
   ```
5. **Copy the entire meta tag** (including the content value)

## 📝 **Step 2: Add Meta Tag to index.html**

Once you provide the meta tag, I'll add it to your `index.html` file in the `<head>` section.

## 🚀 **Step 3: Deploy Changes**

After adding the tag, we'll deploy to Firebase Hosting.

## ✅ **Step 4: Verify in Google Search Console**

1. Go back to Google Search Console
2. Click **"VERIFY"** button
3. Wait for verification to complete (usually instant)

## 🔒 **Step 5: Request Security Review**

After verification is complete:

1. In Google Search Console, go to **"Security Issues"** section
2. If your site is flagged, you'll see an option to **"Request Review"**
3. Click **"Request Review"**
4. Fill out the form:
   - Explain that this is a legitimate business website
   - Mention it's a device selling platform
   - State that you've fixed any issues
5. Submit the request
6. Wait 24-48 hours for Google to review
